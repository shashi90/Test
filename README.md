

# Loktra



## Usage
git pull the code.
Run "npm install"
Run "node app.js"

Reverse Hash
http://localhost:3000/loktratest/reversehash (for local Server)

WebCrawler
http://localhost:3000/loktratest/webcrawler (for local server)
